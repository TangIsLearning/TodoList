# 任务列表空间优化

## 优化目标
解决小屏幕时任务列表项占用空间过大的问题，提升信息密度和用户体验。

## 优化策略

### 📱 屏幕尺寸分层优化

#### 1. 中等屏幕 (769px-1024px)
- **适度紧凑**: 保持良好可读性的同时节省空间
- **减少间距**: 任务间距从 `var(--spacing-md)` 调整为 `var(--spacing-sm)`
- **最小高度**: 140px（相比默认180px更紧凑）

#### 2. 小屏幕 (≤480px)
- **显著紧凑**: 大幅减少空间占用
- **最小高度**: 120px，适配手机屏幕
- **文本优化**: 
  - 标题字体从 `var(--font-size-lg)` 调整为 `var(--font-size-base)`
  - 描述文本限制为2行显示
  - 行高调整为1.4，提升阅读密度

#### 3. 超小屏幕 (≤320px)
- **极简紧凑**: 最大化信息展示
- **最小高度**: 100px，最节省空间
- **激进优化**:
  - 描述文本仅显示1行
  - 标签和按钮尺寸最小化
  - 任务间距缩小至4px

### ✨ 具体优化措施

#### 布局空间优化
```css
/* 小屏幕任务项 */
.task-item {
    padding: var(--spacing-sm) var(--spacing-md);  /* 减少内边距 */
    min-height: 120px;                          /* 降低最小高度 */
}
```

#### 文本内容优化
```css
/* 标题优化 */
.task-title {
    font-size: var(--font-size-base);              /* 更小字体 */
    line-height: 1.3;                           /* 更紧凑行高 */
    margin-bottom: 2px;                          /* 减少下边距 */
}

/* 描述优化 */
.task-description {
    font-size: var(--font-size-sm);               /* 较小字体 */
    line-height: 1.4;                           /* 紧凑行高 */
    -webkit-line-clamp: 2;                        /* 限制2行显示 */
}
```

#### 元素尺寸调整
```css
/* 复选框缩小 */
.task-checkbox {
    width: 18px;                                /* 从20px缩小 */
    height: 18px;
}

/* 标签紧凑化 */
.task-priority,
.task-category {
    padding: 2px 6px;                          /* 减少内边距 */
    font-size: 10px;                             /* 较小字体 */
}

/* 操作按钮优化 */
.task-action-btn {
    padding: 4px 8px;                          /* 更紧凑按钮 */
    font-size: var(--font-size-xs);
}
```

### 📊 空间节省效果

| 屏幕尺寸 | 原高度 | 优化后高度 | 节省空间 | 优化特点 |
|---------|--------|-----------|---------|---------|
| 769px-1024px | 180px | 140px | 22% | 保持可读性 |
| ≤480px | 180px | 120px | 33% | 显著紧凑 |
| ≤320px | 180px | 100px | 44% | 极致紧凑 |
