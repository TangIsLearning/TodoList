TodoList 如何融入你的日常？

💼 工作项目

用分类管理不同项目，为任务设置优先级与截止日期。通过局域网共享，团队内快速同步待办清单。

📚 学习备考

创建周期性学习任务，例如"每天早上8点背单词"。利用标签标记不同科目，通过统计直观看到自己的坚持与成长。

🏡 个人生活

从购物清单到生日提醒，从习惯养成到旅行规划。深色模式适合夜间使用，数据本地存储保障隐私。通过坚果云同步，让手机和电脑上的生活事务保持同步。

高手进阶：这样用更高效

独特设计，让 TodoList 成为你手指的延伸。

📝 自然语言速建任务

按下 Ctrl+Space，像聊天一样输入"完成项目报告 #工作 @1h"，任务便会自动带上标签、分类和截止时间，让记录快如闪电。

🔄 周期性任务，一次创建

对于每周例会、月度还款等重复事项，只需配置一次重复周期，系统便会自动生成后续任务，让规律事务永不遗漏。

☁️ 巧用存储路径上云

将存储路径设为坚果云等网盘的本地同步文件夹，无需额外操作，即可实现数据在多设备间的自动云同步，兼顾本地安全与云端便利。

局域网同步：把数据留在你的设备之间

不依赖云端服务器，多台设备在同一个局域网里直接对话 —— 握手、同步、冲突解决，全部本地完成。

零服务器 P2P

UDP beacon 发现同网节点 + TCP 长连接直连；握手成功后才进入同步通道，不向公网暴露任何端口。

时钟偏移容忍

不同设备的本地时间可能差几秒。冲突解决时 1 秒内视为同时，按节点 ID 字典序裁决，避免抖动来回覆盖。

节点状态可见

在线 / 同步中 / 离线 三态徽章实时显示，断网 30s 内自动标记离线，重连后自动增量同步差异数据。

协议版本校验

每条消息携带协议版本号 v1.0.0；旧版本节点握手会被拒绝并写入事件日志，不会导致误解析。

网络事件可追溯

握手成功 / 失败、协议错误、冲突解决 —— 6 类事件全量落库，按类型过滤，定位问题不靠猜。

断网重连不丢数据

记录 last_sync_at，重连后只拉取增量；本地变更也会在重连瞬间广播给对端，最终一致性收敛。
