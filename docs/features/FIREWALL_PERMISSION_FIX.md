# 防火墙权限问题修复说明

## 问题描述

当应用以普通用户身份运行时，添加防火墙规则会失败，提示"需要管理员权限才能执行此操作"。

## 修复方案

### 方案1: 自动UAC权限提升（推荐）

**实现原理**:
应用会自动检测当前用户权限，如果没有管理员权限，会通过Windows UAC（用户账户控制）请求提升权限。

**使用方法**:
1. 正常启动TodoList应用
2. 点击"开始共享"
3. 如果需要管理员权限，系统会弹出UAC提示框
4. 点击"是"允许提升权限
5. 防火墙规则会自动添加

**优点**:
- 无需手动配置
- 自动检测和处理
- 用户友好

**注意**:
- UAC提示框只会出现一次（添加规则时）
- 删除规则时也会触发UAC提示
- 如果用户点击"否"，规则不会添加，但服务器仍会启动

### 方案2: 使用批处理脚本（备用）

如果自动权限提升失败，可以使用提供的批处理脚本手动配置防火墙。

#### 添加防火墙规则

1. 右键点击 `setup_firewall.bat`
2. 选择"以管理员身份运行"
3. 等待配置完成
4. 显示"配置完成！"表示成功

#### 删除防火墙规则

1. 右键点击 `remove_firewall.bat`
2. 选择"以管理员身份运行"
3. 等待删除完成
4. 显示"删除完成！"表示成功

### 方案3: 手动执行命令（高级用户）

#### PowerShell（管理员）
```powershell
netsh advfirewall firewall add rule name="TodoList P2P Server" dir=in action=allow protocol=TCP localport=5353
```

#### 命令提示符（管理员）
```cmd
netsh advfirewall firewall add rule name="TodoList P2P Server" dir=in action=allow protocol=TCP localport=5353
```

#### 图形界面

1. 按 `Win + I` 打开设置
2. 进入 **网络和 Internet** → **Windows 安全中心**
3. 点击 **防火墙和网络保护**
4. 点击 **允许应用通过防火墙**
5. 点击 **更改设置**
6. 找到并勾选 `python.exe` 或 `TodoList.exe`
7. 勾选 **专用** 和 **公用**
8. 点击 **确定**

### 方案4: 以管理员身份运行应用

最简单的方法是以管理员身份运行应用：

**方法1: 右键菜单**
1. 右键点击应用图标（`main.exe` 或 Python脚本）
2. 选择"以管理员身份运行"

**方法2: 快捷方式**
1. 创建应用的快捷方式
2. 右键点击快捷方式 → 属性
3. 快捷方式标签 → 高级
4. 勾选"以管理员身份运行此程序"
5. 确定 → 确定

## 常见问题

### Q1: 为什么会出现UAC提示？
**A**: UAC（用户账户控制）是Windows的安全机制，当程序尝试执行需要管理员权限的操作时，系统会弹出提示框让用户确认。这是正常的安全措施。

### Q2: 我不想每次都看到UAC提示怎么办？
**A**:
- 方案1: 以管理员身份运行应用（推荐）
- 方案2: 使用批处理脚本一次性配置
- 方案3: 关闭UAC（不推荐，存在安全风险）

### Q3: UAC提示没有出现，但防火墙规则添加失败？
**A**: 可能是UAC被禁用或系统配置问题。建议：
1. 使用批处理脚本手动配置
2. 以管理员身份运行应用
3. 检查Windows防火墙服务是否正常运行

### Q4: 批处理脚本无法运行？
**A**: 检查以下几点：
1. 确保以管理员身份运行
2. 检查杀毒软件是否阻止
3. 检查Windows防火墙服务是否正常运行

### Q5: 防火墙规则已添加，但其他设备仍无法连接？
**A**: 检查以下几点：
1. 确认两台设备在同一网段（如192.168.125.x）
2. 确认P2P服务器正在运行
3. 检查是否有其他安全软件（杀毒软件）阻止连接
4. 尝试增加扫描超时时间
5. 检查路由器是否启用了AP隔离功能

### Q6: 如何永久删除UAC提示？
**A**: 不推荐完全关闭UAC，这会降低系统安全性。如果确实需要：

**Windows 10/11**:
1. 按 `Win + R`，输入 `gpedit.msc`（需要专业版或企业版）
2. 计算机配置 → Windows 设置 → 安全设置 → 本地策略 → 安全选项
3. 找到"用户账户控制: 管理员批准模式下管理员的提升提示行为"
4. 设置为"不提示，直接提升"
5. 重启电脑

**注意**: 关闭UAC会降低系统安全性，仅用于开发和测试环境！

## 技术细节

### UAC权限提升实现

使用Windows API `ShellExecute` 函数的 `runas` 动词请求权限提升：

```python
import ctypes
from ctypes import wintypes

ShellExecute = ctypes.windll.shell32.ShellExecuteW
ShellExecute.argtypes = [
    wintypes.HWND,
    wintypes.LPCWSTR,
    wintypes.LPCWSTR,
    wintypes.LPCWSTR,
    wintypes.LPCWSTR,
    wintypes.INT
]
ShellExecute.restype = wintypes.HINSTANCE

# 使用runas动词请求管理员权限
result = ShellExecute(
    None,
    "runas",
    "netsh",
    command,
    None,
    1  # SW_SHOWNORMAL
)
```

### 返回值处理

- **result > 32**: ShellExecute成功执行
- **result <= 32**: ShellExecute失败（如用户取消UAC）
